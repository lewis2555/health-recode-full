# 健康记录 APP 使用打包指南 📱💊
## 🎯 项目介绍
这是一个专门为记录吃药时间、血压、血糖设计的 Android APP，支持：
- ✅ 记录吃药时间
- ✅ 记录血压数值（收缩压/舒张压）
- ✅ 记录血糖数值
- ✅ 支持拍照保存
- ✅ 查看历史记录列表
## 🚀 三步打包法（超简单！）
### 第一步：上传代码到 GitHub
1. 打开 GitHub 官网（github.com），注册/登录账号
2. 创建一个新仓库（名字随便起，比如 `health-record-app`）
3. 把当前文件夹里的所有文件上传到这个仓库
> 💡 当前文件夹位置：

> ```
### 第二步：自动打包
上传完代码后，GitHub 会**自动开始打包**，不用您动手！
- 点击仓库的 `Actions` 标签页
- 能看到正在运行 `Build Android APK with Buildozer`
- 打包需要大约 **15-20 分钟**，喝杯茶等一等就好 ☕
### 第三步：下载 APK
打包完成后：
1. 进入 `Actions` -> 最新的构建 -> `Artifacts`
2. 点击 `app-apk` 下载压缩包
3. 解压后就能得到 APK 文件
4. 把 APK 文件传到您的安卓手机上，点击安装就可以用了！

## 📸 APP 使用说明
安装打开后：
1. **添加记录**：点击右下角 "+" 按钮
2. **选择类型**：吃药/血压/血糖
3. **填写信息**：自动获取当前时间，您只需要填数值
4. **拍照**：需要的话可以拍照保存
5. **保存**：点击保存就完成了
6. **查看历史**：主界面能看到所有历史记录，可以滑动删除
## ❓ 常见问题
**Q: 打包失败了怎么办？**
A: 重新点一下 `Re-run jobs` 就好了，网络问题很正常
**Q: 安装的时候提示未知来源？**
A: 在手机设置里允许 "来自此来源" 就可以安装了
**Q: 占多大空间？**
A: APK 大约 30-50 MB，非常小
